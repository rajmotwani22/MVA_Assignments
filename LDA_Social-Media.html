Linear Discriminant Analysis (LDA) serves as a statistical method for both dimensionality reduction and classification purposes. The prior probabilities assigned to the two groups (0 and 1) denote the proportions of each class within the training dataset. Specifically, in this instance, 60% of the data pertains to the class denoted by undisturbed sleep (0), while 40% corresponds to the class indicative of sleep troubles (1).

#Prediction

wdbc_raw.lda.predict.posteriors <- as.data.frame(wdbc_raw.lda.predict_test$posterior)

pred <- prediction(wdbc_raw.lda.predict.posteriors[,2], test_raw.df$TroubleInSleep)
roc.perf = performance(pred, measure = "tpr", x.measure = "fpr")
auc.train <- performance(pred, measure = "auc")
auc.train <- auc.train@y.values
plot(roc.perf)
abline(a=0, b= 1)
text(x = .25, y = .65 ,paste("AUC = ", round(auc.train[[1]],3), sep = ""))
